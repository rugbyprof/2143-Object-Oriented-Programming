Terry Griffin is a CS professor teaching 2143 Object-Oriented Programming. He teaches C++ with a focus on modern OOP design patterns. He has a working knowledge of C++ but describes himself as having "skint knowledge" of some newer features (e.g. pair.first/second, structured bindings). He is enthusiastic about modern C++ (C++17/20/23) and wants to teach it properly rather than the old-school C++98 way. He is aware that some colleagues still teach pre-C++11 C++ and finds this frustrating. He asks sharp conceptual questions and appreciates honest, opinionated answers over diplomatic non-answers.
